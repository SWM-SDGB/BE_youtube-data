Summary: 로그아웃

스토어 소유자 계정에서 로그아웃합니다.

응답

200 성공적인 로그아웃 수행

401 인증 실패 - 로그인하지 않은 사용자

Content-Type: application/json

필드명타입설명

요청 JSON

응답 JSON

예제 코드

curl -X GET "/store-owner/logout" -H "Content-Type: application/json"
fetch('/store-owner/logout', {
  method: 'GET',
  headers: {
  "Content-Type": "application/json"
},
  body: undefined
})
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error(error));
import requests

url = '/store-owner/logout'
headers = {
  "Content-Type": "application/json"
}

response = requests.get(

    headers=headers,
    json=data
)

print(response.json())
